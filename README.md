# 🍕 Мобильное приложение «Пиццерия» на .NET MAUI

## 📝 Общее описание
Данное приложение — это кроссплатформенное решение для заказа пиццы, разработанное на фреймворке **.NET MAUI (8.0)**. Проект демонстрирует реализацию полноценного цикла заказа: от выбора позиций в меню до сохранения истории покупок в локальное хранилище.

---

## 🛠 Технологический стек
*   **Framework:** .NET MAUI 8.0
*   **Архитектура:** MVVM (Model-View-ViewModel)
*   **Библиотеки:** 
    *   `CommunityToolkit.Mvvm` — использование Source Generators для реактивности интерфейса.
    *   `System.Text.Json` — сериализация и работа с данными.
*   **Хранение данных:** Локальный JSON-файл `orders.json`.

---

## 🌟 Основные возможности
*   **Интерактивное Меню**: Просмотр списка пицц с описаниями и актуальными ценами.
*   **Премиальный Конструктор**: 
    *   Двухпанельный интерфейс (Split View) для десктопных версий.
    *   Мгновенный пересчет стоимости при изменении размера, ингредиентов или количества.
    *   Визуальная индикация выбранных компонентов с помощью триггеров.
*   **Корзина**: Управление списком выбранных товаров перед оформлением.
*   **История заказов**: Полноценный архив заказов, который подгружается из внешнего JSON-файла.

---

## 📂 Структура проекта
*   **`Models/`** — Классы данных (Pizza, Ingredient, Order).
*   **`ViewModels/`** — Логика экранов и обработка команд пользователя.
*   **`Views/`** — Разметка интерфейса на языке XAML.
*   **`Services/`** — Бизнес-логика: работа с корзиной, меню и JSON-хранилищем.

---

## 🚀 Инструкция по проверке
1.  **Запуск**: Рекомендуется запускать на **Windows Machine**.
2.  **JSON Хранилище**: Файл `orders.json` создается автоматически в корневой папке проекта (рядом с `.sln`).
3.  **Демонстрация**: При оформлении заказа приложение автоматически открывает проводник Windows, указывая на созданный файл JSON.
4.  **Ресурсы**: Изображения находятся в папке `Resources/Images`.
