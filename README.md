Мобильное приложение «Пиццерия» на .NET MAUI
Общее описание
Данное приложение представляет собой кроссплатформенное решение для заказа пиццы, разработанное на фреймворке .NET MAUI (8.0). Приложение позволяет просматривать меню, настраивать параметры пиццы в интерактивном конструкторе, управлять корзиной и просматривать историю заказов, сохраненную локально.
Технологический стек
Framework: .NET MAUI 8.0
Architecture: MVVM (Model-View-ViewModel)
Library: CommunityToolkit.Mvvm (Source Generators для свойств и команд)
Data Format: JSON (System.Text.Json) для хранения истории заказов.
UI/UX: Адаптивная верстка (Split View для десктопных версий), кастомные стили и триггеры.
Основные возможности
Экран меню: Динамический список доступных позиций с ценами "от".
Конструктор пиццы:
Выбор размера с автоматическим пересчетом стоимости (базовая цена × коэффициент).
Добавление дополнительных ингредиентов с визуальной индикацией выбора.
Расчёт итоговой стоимости в реальном времени по формуле: (База * Коэф + Ингредиенты) * Кол-во.
Корзина: Просмотр списка добавленных позиций, управление количеством и удаление товаров.
История заказов: Просмотр всех ранее оформленных заказов. Данные подгружаются из локального JSON-файла.
Технические особенности реализации
MVVM: Логика полностью отделена от представления. Используются ObservableProperty и RelayCommand для реактивного обновления интерфейса.
Dependency Injection: Все сервисы (PizzaService, CartService, OrderService) и ViewModel зарегистрированы в контейнере зависимостей в MauiProgram.cs.
Локальное хранилище: Реализован OrderService, который осуществляет асинхронное чтение и запись истории заказов в файл orders.json.
UX-оптимизация: Для Windows-версии реализован двухпанельный интерфейс в конструкторе, исключающий пустоты по бокам экрана на мониторах с высоким разрешением.
Инструкция для проверки
Запуск: Приложение оптимизировано для запуска на Windows Machine.
JSON Файл: Для удобства демонстрации файл истории заказов orders.json создается автоматически в корневой папке проекта (рядом с .sln).
Авто-открытие: При нажатии кнопки «Оформить заказ» приложение автоматически открывает проводник Windows с выделенным файлом JSON для подтверждения корректности сохранения данных.
Ресурсы: Все изображения пицц находятся в Resources/Images и корректно индексируются сборщиком MauiImage.
Структура проекта
/Models — классы данных (Pizza, Ingredient, Order).
/ViewModels — логика экранов.
/Views — XAML разметка интерфейса.
/Services — бизнес-логика (работа с данными и корзиной).
