# 🍕 Мобильное приложение «Пиццерия» на .NET MAUI

## 📝 Общее описание
Данное приложение — это кроссплатформенное решение для заказа пиццы. Проект выполнен на фреймворке **.NET MAUI 8.0** с использованием архитектуры **MVVM**.

---

## 📸 Руководство пользователя (Screenshots)

### 1. Выбор пиццы
На главном экране представлен широкий ассортимент пицц. Каждая карточка содержит изображение, описание и базовую цену.

### 2. Конструктор заказа
При клике на кнопку «Выбрать» открывается экран настройки. Здесь можно:
* Выбрать размер (Маленькая/Средняя/Большая).
* Добавить дополнительные ингредиенты (цена обновится мгновенно).
* Указать количество штук.

### 3. Корзина и история
После добавления товара он попадает в корзину. При нажатии «Оформить заказ» данные сохраняются в локальный JSON-файл и отображаются в истории.

## 🛠 Технологический стек
*   **Framework:** .NET MAUI 8.0 (MVVM)
*   **Библиотеки:** `CommunityToolkit.Mvvm`, `System.Text.Json`.
*   **Хранилище:** Локальный файл `orders.json`.

## 🚀 Инструкция по проверке
1.  Запустите проект на **Windows Machine**.
2.  Оформите тестовый заказ. Приложение автоматически откроет проводник с JSON-файлом истории.
3.  Перейдите на вкладку **История**, чтобы увидеть сохраненные данные.
