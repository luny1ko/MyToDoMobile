🍕 Мобильное приложение «Пиццерия» на .NET MAUI
📝 Общее описание
Данное приложение — это современное кроссплатформенное решение для заказа пиццы, разработанное с использованием фреймворка .NET MAUI (8.0). Проект демонстрирует навыки построения сложных интерфейсов, работы с локальным хранилищем данных и архитектурным паттерном MVVM.
🛠 Технологический стек
Платформа: .NET MAUI 8.0
Архитектура: MVVM (Model-View-ViewModel)
Библиотеки:
CommunityToolkit.Mvvm — использование Source Generators для реактивности.
System.Text.Json — сериализация данных.
Хранение данных: Локальный JSON-файл orders.json.
🌟 Основные возможности
Интерактивное Меню: Просмотр ассортимента пицц с динамической подгрузкой данных.
Продвинутый Конструктор:
Split-View интерфейс: Оптимизирован под десктопные экраны (картинка слева, настройки справа).
Умный расчет: Мгновенный пересчет цены при выборе размера или ингредиентов.
Визуальный отклик: Акцентная подсветка выбранных компонентов.
Умная Корзина: Управление списком покупок и расчет итоговой суммы.
История Заказов: Автономное хранение истории покупок. Данные сохраняются даже после закрытия приложения.
📂 Структура проекта
Models/ — Сущности данных (Пицца, Ингредиент, Заказ).
ViewModels/ — Логика управления состоянием интерфейса.
Views/ — Разметка экранов на XAML.
Services/ — Сервисы для работы с корзиной, меню и JSON-хранилищем.
🚀 Инструкция по запуску и проверке
Платформа: Рекомендуется запускать на Windows Machine.
История заказов: Файл orders.json создается автоматически в корневой папке проекта.
Демонстрация сохранения: При нажатии «Оформить заказ» приложение автоматически открывает проводник с созданным JSON-файлом.
Изображения: Все графические ресурсы оптимизированы и находятся в папке Resources/Images.
